FoamOps

ESP32 Cheap Yellow Display game prop firmware

Flash Firmware

Requires Chrome or Edge on desktop. Connect your CYD via USB before clicking.

  1. Connect your CYD (ESP32-2432S028R) to your PC via USB
  2. Click Install FoamOps above
  3. Select the correct COM port when prompted
  4. Hold the BOOT button on the back of the board until the progress bar starts moving — the screen may flash on and off until you do
  5. Release BOOT and wait for flashing to complete (~30 seconds)
  6. The device will reboot automatically into the game mode selector
The installer will show an Erase device checkbox. Tick it for a clean first-time install. Leave it unticked when updating firmware if you want to keep your saved settings (brightness, sound, ESP-NOW pairs).

Game Modes

Flux Two-team domination timer. Tap your team colour to capture.
Defuse Bomb countdown with code or wire cut puzzle to defuse.
HD2 Spawn Helldivers-style stratagem sequence for respawn gate.
Spawn Count Track team lives. Count up or count down from a set number.
Respawn Waves Looping wave timer. Shows RESPAWN when the wave hits zero.
Settings Brightness, sound, display invert, ESP-NOW and board MAC.

Required Hardware

  • ESP32-2432S028R (Cheap Yellow Display / CYD)
  • USB-C cable for flashing
  • 3W 4Ω JST speaker (optional, for sound effects)
  • Battery — single 18650 or USB power bank recommended
